# Security Policy

## 报告安全漏洞

如果您发现本项目存在安全漏洞，请**不要**通过公开的 Issue 报告。

请通过以下方式私密报告：

- 📧 邮件：[security@tencent.com](mailto:security@tencent.com)
- 主题格式：`[SSM-SDK-Python] 安全漏洞报告 - 简要描述`

我们会在 **48 小时内**确认收到报告，并在 **7 个工作日内**提供初步评估。

## 报告内容

请在报告中包含以下信息：

- 漏洞类型和影响范围
- 复现步骤（尽可能详细）
- 受影响的版本
- 可能的修复建议（如有）

## 安全最佳实践

使用本 SDK 时，请遵循以下安全建议：

1. **优先使用 CAM 角色认证**（`SsmAccount.with_cam_role()`），避免在代码中硬编码 AK/SK
2. **不要将凭据提交到版本控制系统**，使用环境变量或配置文件管理敏感信息
3. **定期轮转凭据**，利用 SSM 的自动轮转功能
4. **限制 API 密钥权限**，遵循最小权限原则
5. **及时更新 SDK 版本**，获取最新的安全修复
